树莓派更新源使用阿里云镜像

Jan 18, 2019
Technology

备份原始 apt 更新源列表

sudo cp /etc/apt/sources.list /etc/apt/sources.list.bak

替换 raspbian.raspberrypi.org 为 mirrors.aliyun.com/raspbian

sudo sed -i 's/raspbian.raspberrypi.org/mirrors.aliyun.com\/raspbian/g' \
  /etc/apt/sources.list

更新源

sudo apt update
